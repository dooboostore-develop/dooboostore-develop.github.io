simple-boot-front

Single Page Application framework for web development

웹 개발을 위한 단일 페이지 애플리케이션 프레임워크

Installation

Install the package and its dependencies:

패키지와 의존성을 설치하세요:

npm 
npm install @dooboostore/simple-boot-front reflect-metadata
pnpm 
pnpm add @dooboostore/simple-boot-front reflect-metadata
Quick Start CLI 
npm init @dooboostore/simple-boot-front projectname

Features

SPA Framework

Complete Single Page Application framework for modern web development

현대적인 웹 개발을 위한 완전한 단일 페이지 애플리케이션 프레임워크

Component System

Decorator-based component system with template and style support

템플릿과 스타일을 지원하는 데코레이터 기반 컴포넌트 시스템

Router System

Flexible routing with path parameters and route guards

경로 매개변수와 라우트 가드를 지원하는 유연한 라우팅

Dependency Injection

Built on simple-boot core with full DI and AOP support

완전한 DI와 AOP를 지원하는 simple-boot 코어 기반

Template Engine

Powered by dom-render for efficient DOM manipulation

효율적인 DOM 조작을 위한 dom-render 기반

Script System

Custom script system for reusable template logic

재사용 가능한 템플릿 로직을 위한 커스텀 스크립트 시스템

Quick Start

Basic Setup

Create a simple SPA with routing:

라우팅을 포함한 간단한 SPA를 생성하세요:

pages/home.ts 
@Sim
@Component({
    template: '''
Home Page
'''
})
export class Home {
    
}
pages/user.ts 
@Sim
@Component({
    template: '''
User Page
'''
})
export class User {
    
}
index.ts 
import template from '''./index.html'''
import style from '''./index.css'''

@Sim
@Router({
  path: '''''',
  route: {
    '''/''': Home,
    '''/user''': User
  }
})
@Component({
  template,
  styles: [style]
})
export class Index implements RouterAction {
  child?: any;
  
  async canActivate(url: any, module: any) {
    this.child = module;
  }
}

const config = new SimFrontOption(window).setUrlType(UrlType.hash);
const simpleApplication = new SimpleBootFront(Index, config);
simpleApplication.run();
