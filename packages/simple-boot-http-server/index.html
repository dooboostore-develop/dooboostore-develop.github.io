simple-boot-http-server

Lightweight and powerful HTTP web server framework for Node.js

Node.js를 위한 경량이면서 강력한 HTTP 웹 서버 프레임워크

Installation

Install the package and its dependencies:

패키지와 의존성을 설치하세요:

npm 
npm install @dooboostore/simple-boot-http-server reflect-metadata
pnpm 
pnpm add @dooboostore/simple-boot-http-server reflect-metadata
Quick Start CLI 
npm init @dooboostore/simple-boot-http-server projectname

Features

Declarative Routing

Use decorators (@Router, @Route) to map URLs to classes and methods

데코레이터(@Router, @Route)를 사용하여 URL을 클래스와 메서드에 매핑

HTTP Method Mapping

Decorators for all standard HTTP methods (@GET, @POST, @PUT, @DELETE, etc.)

모든 표준 HTTP 메서드를 위한 데코레이터(@GET, @POST, @PUT, @DELETE 등)

Request/Response Handling

Unified RequestResponse object to manage HTTP transactions

HTTP 트랜잭션을 관리하는 통합된 RequestResponse 객체

Middleware/Filters

Intercept requests and responses for cross-cutting concerns

횡단 관심사를 위한 요청 및 응답 가로채기

Global Exception Handling

Centralized error handling using @Sim advice classes

@Sim advice 클래스를 사용한 중앙집중식 오류 처리

Session Management

Built-in session management with customizable storage providers

사용자 정의 저장소 제공자를 지원하는 내장 세션 관리

HTTPS Support

Configure the server to run over HTTPS and HTTP/2

HTTPS 및 HTTP/2를 통해 실행되도록 서버 구성

Endpoints

Define logic for different stages of request lifecycle

요청 생명주기의 다양한 단계에 대한 로직 정의

Server Setup

Bootstrap your HTTP server with the framework:

프레임워크로 HTTP 서버를 부트스트랩하세요:

Server Setup 
import { SimpleApplication } from '''@dooboostore/simple-boot''';
import { SimpleBootHttpServer, HttpServerOption } from '''@dooboostore/simple-boot-http-server''';

// Create your main router
@Sim
@Router({ 
    path: '''''', 
    route: {
        '''/api''': ItemApi,
        // Add more routes here
    }
})
class MainRouter {}

// Configure and start the server
const httpOption = new HttpServerOption({
    port: 3000,
    host: '''localhost'''
});

const app = new SimpleApplication({ rootRouter: MainRouter });
const server = new SimpleBootHttpServer(app, httpOption);

server.run().then(() => {
    console.log('''Server is running on http://localhost:3000''');
});

            rr.resStatusCode(401);
            return { error: '''Not authenticated''' };
        }
        
        let visitCount = session.visitCount || 0;
        visitCount++;
        session.visitCount = visitCount;
        
        return {
            userId: session.userId,
            username: session.username,
            loginTime: session.loginTime,
            visitCount: visitCount
        };
    }

    @Route({ path: '''/logout''' })
    @POST({ res: { contentType: Mimes.ApplicationJson } })
    async logout(rr: RequestResponse) {
        const session = await rr.reqSession();
        
        // Clear session data
        Object.keys(session).forEach(key => delete session[key]);
        
        return { success: true, message: '''Logged out successfully''' };
    }
}
