core-node

Node.js utilities and file system operations

Node.js 유틸리티 및 파일 시스템 작업

Installation

Install the package and its peer dependency:

패키지와 필수 의존성을 설치하세요:

npm 
npm install @dooboostore/core-node @dooboostore/core
pnpm 
pnpm add @dooboostore/core-node @dooboostore/core

Features

File System Operations

Comprehensive file operations with File class and utility functions

Advanced buffer handling with Base64 support and encoding options

Path Management

Cross-platform path utilities for file and directory operations

Memory Utilities

Memory management and monitoring utilities for Node.js applications

Node.js 애플리케이션을 위한 메모리 관리 및 모니터링 유틸리티

Process Management

Process lifecycle management and event handling utilities

Node.js 런타임 환경에 특별히 설계됨
