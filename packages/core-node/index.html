Path Utilities

Handle cross-platform path operations:

크로스 플랫폼 경로 작업을 처리하세요:

Path Utilities 
import { FileUtils } from '@dooboostore/core-node/file/FileUtils';

// Join path segments
const fullPath = FileUtils.path(['documents', 'images', 'photo.jpg']);
// Result: 'documents/images/photo.jpg' (Unix) or 'documents\\images\\photo.jpg' (Windows)

// Array of path segments
const pathSegments = ['home', 'user', 'projects', 'myapp'];
const projectPath = FileUtils.path(pathSegments);

// Single path string
const singlePath = FileUtils.path('./src/components/Button.tsx');

Asynchronous Operations

Handle asynchronous file operations efficiently:

비동기 파일 작업을 효율적으로 처리하세요:

Asynchronous Operations 
import { FileUtils } from '@dooboostore/core-node/file/FileUtils';

async function processFiles() {
  try {
    // Create directory asynchronously
    const dirPath = await FileUtils.mkdir('./temp/processing');
    
    // Read file asynchronously
    const content = await FileUtils.read('./input.txt', { 
      option: { encoding: 'utf-8' } 
    });
    
    // Process content and write to new file
    const processedContent = content.toString().toUpperCase();
    const buffer = Buffer.from(processedContent);
    
    const outputFile = await FileUtils.writeFile(buffer, {
      path: './temp/processing/output.txt',
      originalName: 'processed-input.txt'
    });
    
    console.log('Processing complete:', outputFile.path);
    
  } catch (error) {
    console.error('File processing error:', error);
  }
}

API Reference

FileUtils Namespace

File Class

Object-oriented file wrapper with methods for file operations

파일 작업을 위한 메서드를 포함한 객체 지향 파일 래퍼

  • constructor(data: {path: string, originalName?: string})
  • copy(newPath: string): Promise<void>
  • move(newPath: string): Promise<void>
  • rename(newName: string): Promise<void>
  • delete(): Promise<void>
  • updateStats(): Promise<void>

Utility Functions

Static utility functions for common file operations

일반적인 파일 작업을 위한 정적 유틸리티 함수

  • writeFile(buffer: Buffer, config): Promise<File>
  • existesSync(path: PathParamType): boolean
  • mkdir(path: PathParamType): Promise<string>
  • readSync(path: PathParamType): Buffer
  • write(data: string | Buffer, config): string
  • deleteSync(path: PathParamType): void

ConvertUtils Namespace

String Conversion

Convert between strings and buffers with encoding support

인코딩 지원으로 문자열과 버퍼 간 변환

  • toString(data: string | Buffer, config?): string
  • toBuffer(data: string, config?): Buffer

Encoding Options

Support for various buffer encodings including Base64

Base64를 포함한 다양한 버퍼 인코딩 지원

  • encoding: BufferEncoding - utf-8, base64, hex, etc.
  • Automatic Base64 data URL prefix removal
