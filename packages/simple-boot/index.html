simple-boot

Lightweight framework with DI and AOP support

DI, AOP를 지원하는 경량 프레임워크

Installation

Install the package and its peer dependency:

패키지와 필수 의존성을 설치하세요:

npm 
npm install @dooboostore/simple-boot reflect-metadata
pnpm 
pnpm add @dooboostore/simple-boot reflect-metadata
Quick Start CLI 
npm init @dooboostore/simple-boot projectname

Features

Object Management

Dependency Injection (DI) and Lifecycle callbacks

의존성 주입(DI)과 생명주기 콜백

Aspect-Oriented Programming

AOP support with @Before, @After, @Around decorators

@Before, @After, @Around 데코레이터를 통한 AOP 지원

Exception Handling

Powerful global and local exception handling

강력한 전역 및 지역 예외 처리

Flexible Router System

Class-based and method-based routing

클래스 기반 및 메서드 기반 라우팅

Intent Event System

Decoupled communication with Intent objects

Intent 객체를 통한 분리된 통신

Method-level Caching

Built-in caching with TTL and dynamic keys

TTL과 동적 키를 지원하는 내장 캐싱

Data Validation

Property validation with decorators

데코레이터를 통한 속성 검증

Quick Start

Basic Usage

Here's a simple example to get you started:

시작하기 위한 간단한 예제입니다:

Basic Usage 
import 'reflect-metadata';
import { SimpleApplication, Sim, Router } from '@dooboostore/simple-boot';

@Sim // Decorator can be used without parentheses
class Greeter {
  say() {
    console.log('Hello, Simple Boot!');
  }
}

@Sim
@Router({ path: '', route: { '/user': Greeter } })
class AppRouter {}

const app = new SimpleApplication({ rootRouter: AppRouter });
app.run();

// Get instance and call method
const greeter = app.sim(Greeter);
greeter?.say();

// Or, route to the module and execute
app.routing('/user').then(routerModule => {
    routerModule.getModuleInstance<Greeter>()?.say();
});

// Singleton (Default) - Same instance every time
@Sim({
  scope: Lifecycle.Singleton
})
class DatabaseService {
  private connection = 'db-connection';
  
  getConnection() {
    return this.connection;
  }
}

// Transient - A new instance is created on each request
@Sim({
  scope: Lifecycle.Transient
})
class RequestHandler {
  private requestId = Math.random();
  
  getId() {
    return this.requestId;
  }
}

Core Concepts

@Sim

Core decorator that marks a class for management by the Simple Boot container

Simple Boot 컨테이너에서 관리할 클래스를 표시하는 핵심 데코레이터

SimpleApplication

Application entry point that initializes and manages the DI container

DI 컨테이너를 초기화하고 관리하는 애플리케이션 진입점

@Router & @Route

Routing system that maps URL paths to classes or methods

URL 경로를 클래스나 메서드에 매핑하는 라우팅 시스템

@Before/@After/@Around

AOP decorators for intercepting method calls

메서드 호출을 가로채는 AOP 데코레이터

Intent System

Event system for decoupled communication between components

컴포넌트 간 분리된 통신을 위한 이벤트 시스템

@Cache

Method-level caching with TTL and dynamic key support

TTL과 동적 키를 지원하는 메서드 레벨 캐싱
