core

TypeScript/JavaScript 프로젝트를 위한 포괄적인 유틸리티 라이브러리

패키지를 설치하세요:

npm 
npm install @dooboostore/core
pnpm 
pnpm add @dooboostore/core
yarn 
yarn add @dooboostore/core

Features

Rich Utilities

Vast collection of utilities for arrays, objects, strings, dates, and math operations

배열, 객체, 문자열, 날짜, 수학 연산을 위한 방대한 유틸리티 컬렉션

Quick Start

Array Utilities

Powerful array manipulation functions:

강력한 배열 조작 함수들:

Array Utilities 
import { ArrayUtils } from '@dooboostore/core/array/ArrayUtils';

const myArray = [1, 2, 3, 4, 5];

// Shuffle the array
const shuffled = ArrayUtils.toShuffle(myArray);
console.log(shuffled);

// Pick a random element
const randomItem = ArrayUtils.pick(myArray);
console.log(randomItem); // e.g., 3

// Pick 3 random unique elements
const randomItems = ArrayUtils.pick(myArray, 3);
console.log(randomItems); // e.g., [5, 1, 3]

// Create 2D array
const grid = ArrayUtils.create2DArray(3, 3, 0);
console.log(grid); // [[0,0,0], [0,0,0], [0,0,0]]

// Set operations
const arr1 = [1, 2, 3, 4];
const arr2 = [3, 4, 5, 6];
const intersection = ArrayUtils.intersection(arr1, arr2);
console.log(intersection); // [3, 4]

Date Utilities

Comprehensive date formatting and manipulation:

포괄적인 날짜 포맷팅과 조작:

Date Utilities 
import { DateUtils } from '@dooboostore/core/date/DateUtils';

const now = new Date();

// Format a date
const formatted = DateUtils.format(now, 'yyyy-MM-dd HH:mm:ss');
console.log(formatted); // "2025-06-28 10:30:00"

// Get relative time
const fiveMinutesAgo = new Date(Date.now() - 5 * 60 * 1000);
const relative = DateUtils.relativeTime(fiveMinutesAgo);
console.log(relative); // "5 minutes ago"

// Date calculations
const tomorrow = DateUtils.addDays(now, 1);
const nextWeek = DateUtils.addWeeks(now, 1);
const nextMonth = DateUtils.addMonths(now, 1);

// Compare dates
const isAfter = DateUtils.isAfter(tomorrow, now);
console.log(isAfter); // true

HTTP Client

Flexible HTTP client with JSON and XML support:

JSON과 XML을 지원하는 유연한 HTTP 클라이언트:

HTTP Client 
import { HttpJsonFetcher, HttpResponseError } from '@dooboostore/core/fetch/HttpJsonFetcher';

const jsonFetcher = new HttpJsonFetcher<any, any>();

async function fetchData() {
  try {
    // GET request for JSON data
    const user = await jsonFetcher.get({
      target: `https://jsonplaceholder.typicode.com/users/1`
    });
    console.log('Fetched User:', user);

    // POST request with JSON body
    const newProduct = await jsonFetcher.postJson({
      target: `https://dummyjson.com/products/add`,
      config: {
        fetch: {
          body: { title: 'New Awesome Gadget', price: 199.99 },
        },
      },
    });
    console.log('Created Product:', newProduct);

  } catch (error: any) {
    if (error instanceof HttpResponseError) {
      console.error('HTTP Error:', error.message, error.response?.status);
    } else {
      console.error('Generic Error:', error);
    }
  }
}
