Reactivity Flow

How state changes propagate to the DOM

상태 변경이 DOM으로 전파되는 방식

sequenceDiagram
    participant User
    participant State
    participant Proxy
    participant Messenger
    participant Operator
    participant DOM

    User->>State: state.count++
    State->>Proxy: set trap triggered
    Proxy->>Proxy: Compare old vs new value
    alt Value Changed
        Proxy->>Messenger: notify(property, newValue)
        Messenger->>Operator: processUpdate()
        Operator->>Operator: Parse template expression
        Operator->>DOM: Update textContent/attribute
        DOM-->>User: Visual update
    else No Change
        Proxy-->>User: Skip update
    end

Component Lifecycle

ComponentBase lifecycle hooks execution order

ComponentBase 생명주기 훅 실행 순서

sequenceDiagram
    participant App
    participant Component
    participant DomRender
    participant DOM

    App->>Component: new MyComponent()
    Component->>Component: constructor()
    Component->>DomRender: Initialize proxy
    DomRender->>Component: OnInitRender()
    Component->>Component: Setup initial state
    DomRender->>DOM: Render template
    DomRender->>Component: OnCreateRendered()
    Component->>Component: Query DOM elements
    Component->>Component: Attach event listeners
    
    Note over Component,DOM: Component is active
    
    App->>Component: destroy()
    Component->>Component: OnDestroyRender()
    Component->>Component: Cleanup resources
    DomRender->>DOM: Remove elements

Getting Started

Quick start with basic examples

Basic Reactive Binding

Create a reactive state object that automatically updates the DOM

DOM을 자동으로 업데이트하는 반응형 상태 객체 생성

import { DomRender } from '@dooboostore/dom-render/DomRender';

// Create a container element
const container = document.createElement('div');
container.innerHTML = `
  <h3>Title: <span>\</span></h3>
  <p>Count: <span>\</span></p>
  <button id="increment">Increment</button>
`;
document.body.appendChild(container);

// Initialize with reactive state
let state = {
  title: 'Hello DomRender!',
  count: 0
};

const result = new DomRender({
  rootObject: state,
  target: container,
  config: { window }
});
state = result.rootObject; // Get proxied state

// Reactive updates - DOM updates automatically!
document.getElementById('increment').addEventListener('click', () =&gt; {
  state.count++; // UI updates instantly
});

Array Binding

Reactive array manipulation with automatic list rendering

자동 리스트 렌더링을 통한 반응형 배열 조작

import { DomRender } from '@dooboostore/dom-render/DomRender';

const container = document.createElement('div');
container.innerHTML = `
  <button id="add-item">Add Item</button>
  <ul>
    <li dr-for-of="@𝒕𝒉𝒊𝒔@.items" dr-option-item-variable-name="item">
      \ - $\
      <button dr-owner-variable-path="this" dr-event-click="@𝒕𝒉𝒊𝒔@.removeItem(#item#)">Remove</button>
    </li>
  </ul>
  <p>Total: <strong>$\</strong></p>
`;
document.body.appendChild(container);

let state = {
  items: [
    { name: 'Apple', price: 1.2 },
    { name: 'Banana', price: 0.8 }
  ],
  get total() {
    return this.items.reduce((sum, item) =&gt; sum + item.price, 0).toFixed(2);
  },
  removeItem(item) {
    const index = this.items.indexOf(item);
    if (index &gt; -1) this.items.splice(index, 1);
  }
};

const result = new DomRender({
  rootObject: state,
  target: container,
  config: { window }
});
state = result.rootObject;

document.getElementById('add-item').addEventListener('click', () =&gt; {
  state.items.push({
    name: `Item </h3>
      <div class="count-display">\</div>
      <button class="increment">+</button>
      <button class="decrement">-</button>
    </div>
  `;
}

// Register and use component
import { drComponent } from '@dooboostore/dom-render/components';

drComponent(CounterComponent, 'counter-component');

// Use in HTML
// <counter-component initial-value="10" data-label="My Counter"></counter-component>

LIFECYCLE HOOKS 생명주기 훅
  • OnInitRender(): void Called once when component is initialized, before first render 컴포넌트 초기화 시 첫 렌더링 전에 한 번 호출
  • OnCreateRenderData(params: OnCreateRenderDataParams): void Called before render, can modify render data 렌더링 전 호출, 렌더 데이터 수정 가능
  • OnCreateRendered(): void Called after first render complete, DOM elements available 첫 렌더링 완료 후 호출, DOM 요소 사용 가능
  • OnRawSetRendered(render: Render, otherData: OnRawSetRenderedOtherData): void Called after each re-render 매 재렌더링 후 호출
  • OnChangeAttrRender(attrName: string, newValue: any, otherData: OtherData): void Called when attribute changes 속성 변경 시 호출
  • OnDestroyRender(params: OnDestroyRenderParams): void Called when component is destroyed, cleanup resources 컴포넌트 파괴 시 호출, 리소스 정리
DECORATORS 데코레이터
  • @attribute(name?: string | {name?, converter?}) Bind HTML attribute to component property, supports custom converter HTML 속성을 컴포넌트 속성에 바인딩, 커스텀 변환기 지원
  • @query(selector | {selector, convert?, onDestroy?, refreshIntervalTime?}) Query and bind DOM element to property, auto-refresh support DOM 요소 쿼리 및 속성에 바인딩, 자동 새로고침 지원
  • @event(eventName, selector, {onDestroy?, refreshIntervalTime?}?) Declarative event listener binding with auto-cleanup 자동 정리 기능이 있는 선언적 이벤트 리스너 바인딩

ComponentSet

Create lightweight component instances without custom elements

커스텀 요소 없이 경량 컴포넌트 인스턴스 생성

import { ComponentSet } from '@dooboostore/dom-render/components/ComponentSet';
import { DomRender } from '@dooboostore/dom-render/DomRender';

// Simple counter class
class Counter {
  count = 0;
  increment() { this.count++; }
  decrement() { this.count--; }
}

// Create component instances
const counters = [
  new ComponentSet(new Counter(), {
    template: `
      <div>
        <p>Count: \</p>
        <button dr-owner-variable-path="this" dr-event-click="@𝒕𝒉𝒊𝒔@.increment()">+</button>
        <button dr-owner-variable-path="this" dr-event-click="@𝒕𝒉𝒊𝒔@.decrement()">-</button>
      </div>
    `
  }),
  new ComponentSet(new Counter(), {
    template: `
      <div>
        <p>Count: \</p>
        <button dr-owner-variable-path="this" dr-event-click="@𝒕𝒉𝒊𝒔@.increment()">+</button>
        <button dr-owner-variable-path="this" dr-event-click="@𝒕𝒉𝒊𝒔@.decrement()">-</button>
      </div>
    `
  })
];

// Render in loop
const container = document.createElement('div');
container.innerHTML = `
  <div dr-for-of="@𝒕𝒉𝒊𝒔@.counters">
    <div dr-this="#it#"></div>
  </div>
`;

const state = new DomRender({
  rootObject: { counters },
  target: container,
  config: { window }
}).rootObject;
