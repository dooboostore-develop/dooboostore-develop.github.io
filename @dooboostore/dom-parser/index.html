Overview

@dooboostore/dom-parser provides a complete DOM implementation for Node.js environments, enabling server-side HTML parsing and manipulation with browser-compatible APIs.

@dooboostore/dom-parser는 Node.js 환경을 위한 완전한 DOM 구현을 제공하여, 브라우저 호환 API로 서버 사이드 HTML 파싱과 조작을 가능하게 합니다.

Basic Usage

Parse HTML and access DOM elements using familiar browser APIs

친숙한 브라우저 API를 사용하여 HTML 파싱 및 DOM 요소 접근

import { DomParser } from
'@dooboostore/dom-parser';

// Parse HTML string
const html = `
〈!DOCTYPE html〉
〈html〉

〈head〉
    〈title〉Test Page〈/title〉
〈/head〉

〈body〉
    〈div id="app"〉
        〈h1〉Hello World〈/h1〉
        〈p class="content"〉This is a test paragraph.〈/p〉
    〈/div〉
〈/body〉

〈/html〉
`;

// Create DOM parser instance
const parser = new DomParser(html);

// Access document and window objects
const document = parser.document;
const window = parser.window;

// Use standard DOM APIs
const title = document.querySelector('title');
console.log(title?.textContent); // "Test Page"

const app = document.getElementById('app');
console.log(app?.innerHTML); // Contains the div content

const paragraphs = document.querySelectorAll('p.content');
console.log(paragraphs.length); // 1

Server-Side Rendering (SSR)

Set up global DOM environment for SSR applications

SSR 애플리케이션을 위한 글로벌 DOM 환경 설정

import { DomParser } from
'@dooboostore/dom-parser';
import fs from 'fs';

// Load HTML template
const templateHtml = fs.readFileSync('template.html', 'utf8');
const parser = new DomParser(templateHtml);

// Set up global DOM for SSR
global.document = parser.document;
global.window = parser.window;

// Now your components can use DOM APIs on the server
function renderComponent() {
    const element = document.createElement('div');
    element.textContent = 'Server-rendered content';
    document.body.appendChild(element);

    return document.documentElement.outerHTML;
}

const renderedHtml = renderComponent();
console.log(renderedHtml);

HTML Processing & Transformation

Process and transform HTML content programmatically

프로그래밍 방식으로 HTML 콘텐츠 처리 및 변환

import { DomParser } from
'@dooboostore/dom-parser';

function processHtml(html: string) {
    const parser = new DomParser(html);
    const document = parser.document;

    // Modify the title
    const title = document.querySelector('title');
    if (title) {
        title.textContent = 'Processed - ' + title.textContent;
    }

    // Add meta tags
    const head = document.querySelector('head');
    const metaDescription = document.createElement('meta');
    metaDescription.setAttribute('name', 'description');
    metaDescription.setAttribute('content', 'Processed by dom-parser');
    head?.appendChild(metaDescription);

    // Add CSS class to all paragraphs
    const paragraphs = document.querySelectorAll('p');
    paragraphs.forEach(p => {
        p.classList.add('processed');
    });

    // Wrap content in a container
    const body = document.body;
    const wrapper = document.createElement('div');
    wrapper.className = 'content-wrapper';

    // Move all body children to wrapper
    while (body.firstChild) {
        wrapper.appendChild(body.firstChild);
    }
    body.appendChild(wrapper);

    return document.documentElement.outerHTML;
}

const originalHtml = `
〈!DOCTYPE html〉
〈html〉
    〈head〉〈title〉Original〈/title〉〈/head〉
    〈body〉
        〈p〉First paragraph〈/p〉
        〈p〉Second paragraph〈/p〉
    〈/body〉
〈/html〉
`;

const processedHtml = processHtml(originalHtml);
console.log(processedHtml);

Web Scraping & Data Extraction

Extract structured data from HTML content

HTML 콘텐츠에서 구조화된 데이터 추출

import { DomParser } from
'@dooboostore/dom-parser';

function extractProductData(html: string) {
const parser = new DomParser(html);
const document = parser.document;

// Extract product information
const products = [];
const productElements = document.querySelectorAll('.product-item');

productElements.forEach(element => {
const name = element.querySelector('.product-name')?.textContent?.trim();
const price = element.querySelector('.price')?.textContent?.trim();
const image = element.querySelector('img')?.getAttribute('src');
const rating = element.querySelectorAll('.star.filled').length;

if (name && price) {
    products.push({
        name,
        price,
        image,
        rating
    });
}
});

return products;
}
