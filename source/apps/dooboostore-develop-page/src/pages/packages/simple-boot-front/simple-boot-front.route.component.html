simple-boot-front

Single Page Application framework for web development

웹 개발을 위한 단일 페이지 애플리케이션 프레임워크

Installation

Install the package and its dependencies:

패키지와 의존성을 설치하세요:

npm 
npm install @dooboostore/simple-boot-front reflect-metadata
pnpm 
pnpm add @dooboostore/simple-boot-front reflect-metadata
Quick Start CLI 
npm init @dooboostore/simple-boot-front projectname
cd projectname
npm start

Quick Start

Basic Setup

Create a simple SPA with routing:

라우팅을 포함한 간단한 SPA를 생성하세요:

pages/home.ts 
@Sim
@Component({
    template: '
Home Page
'
})
export class Home {
    
}
pages/user.ts 
@Sim
@Component({
    template: '
User Page
'
})
export class User {
    
}
index.ts 
import template from './index.html'
import style from './index.css'

@Sim
@Router({
  path: '',
  route: {
    '/': Home,
    '/user': User
  }
})
@Component({
  template,
  styles: [style]
})
export class Index implements RouterAction {
  child?: any;
  
  async canActivate(url: any, module: any) {
    this.child = module;
  }
}

const config = new SimFrontOption(window).setUrlType(UrlType.hash);
const simpleApplication = new SimpleBootFront(Index, config);
simpleApplication.run();

Component System

Create reusable components with templates and styles:

템플릿과 스타일을 포함한 재사용 가능한 컴포넌트를 생성하세요:

template.html 
<h1>${this.title}</h1>
<div>#innerHTML#</div>
component.ts 
import template from './template.html'
import style from './style.css'

@Sim
@Component({
  selector: 'my-component', // default: class name lowercase
  template,
  styles: [style]
})
export class MyComponent {
  public title = 'Hello Component'
  
  public setData(title: string) {
    this.title = title;
  }
}
Usage in template 
<my-component></my-component>

<!-- With data binding -->
<my-component dr-set="$myComponent.setData('Dynamic Title')">
  <ul>
    <li>Custom content</li>
  </ul>
</my-component>

Advanced Routing

Handle dynamic routes with path parameters:

경로 매개변수를 사용한 동적 라우팅을 처리하세요:

@Sim
@Router({
    path: '',
    route: {
        '/': Home,
        '/user': User,
        '/user/{id}': UserDetail
    }
})
@Component({
    template,
    styles: [style]
})
export class AppRouter implements RouterAction {
    child?: any;
    
    canActivate(url: any, module: any): void {
        this.child = module;
    }
}

// Access path parameters
@Sim
export class UserDetail {
    constructor(private routerManager: RouterManager) {
        // Get path data: /user/123 -> { id: '123' }
        const userId = routerManager.activeRouterModule.pathData.id;
        console.log('User ID:', userId);
    }
}

Advanced Features

Script System

Create reusable template scripts for common functionality:

공통 기능을 위한 재사용 가능한 템플릿 스크립트를 생성하세요:

@Sim({scheme: 'i18nScript'})
@Script({
    name: 'i18n'
})
export class I18nScript extends ScriptRunnable {
    public language?: Language;
    
    constructor(public i18nService: I18nService) {
        super();
        i18nService.subject.subscribe(it => {
            this.language = it;
            this.render();  // Re-render components using this script
        })
    }
    
    run(key: string): any {
        return this.language?.defaultData?.[key] ?? key;
    }
}

// Usage in templates
// <div>${$scripts.i18n('Get Locale JSON')}</div>
// <div dr-if="$scripts.i18n('Get Locale JSON') === 'wow'"> is wow</div>

Lifecycle Hooks

Hook into component lifecycle events:

컴포넌트 생명주기 이벤트에 연결하세요:

@Sim
@Component({
    template: '<div>Lifecycle Component</div>'
})
export class LifecycleComponent implements OnInit, OnDestroy, OnChangedRender {
    
    @PostConstruct
    postConstruct(projectService: ProjectService) {
        console.log('Post construct with DI');
    }
    
    onInit(): void {
        console.log('Component initialized');
    }
    
    onChangedRender(): void {
        console.log('Component re-rendered');
    }
    
    onDestroy(): void {
        console.log('Component destroyed');
    }
}

AOP Support

Use Aspect-Oriented Programming for cross-cutting concerns:

횡단 관심사를 위한 관점 지향 프로그래밍을 사용하세요:

@Sim
@Component({
    template: '<button dr-event-click="fire($event, $view)">Click me</button>'
})
export class AopComponent {
    data = 0;
    
    fire($event: MouseEvent, view: View<Element>) {
        console.log('fire method');
        this.data = RandomUtils.random(0, 100);
    }

    @Before({property: 'fire'})
    before(obj: any, protoType: Function) {
        console.log('before fire method', obj, protoType);
    }

    @After({property: 'fire'})
    after(obj: any, protoType: Function) {
        console.log('after fire method', obj, protoType);
    }
}

Core Concepts

@Component

Decorator for creating reusable UI components with templates and styles

템플릿과 스타일을 포함한 재사용 가능한 UI 컴포넌트 생성 데코레이터

@Router

Routing system that maps URLs to components with path parameters

경로 매개변수를 포함하여 URL을 컴포넌트에 매핑하는 라우팅 시스템

@Script

Custom script system for reusable template logic and utilities

재사용 가능한 템플릿 로직과 유틸리티를 위한 커스텀 스크립트 시스템

SimpleBootFront

Main application class that bootstraps the SPA framework

SPA 프레임워크를 부트스트랩하는 메인 애플리케이션 클래스

DOM Render Engine

Efficient DOM manipulation and template rendering system

효율적인 DOM 조작과 템플릿 렌더링 시스템

Lifecycle Management

Component lifecycle hooks for initialization, rendering, and cleanup

초기화, 렌더링, 정리를 위한 컴포넌트 생명주기 훅
