simple-boot-http-server

Lightweight and powerful HTTP web server framework for Node.js

Node.js를 위한 경량이면서 강력한 HTTP 웹 서버 프레임워크

Installation

Install the package and its dependencies:

패키지와 의존성을 설치하세요:

npm install @dooboostore/simple-boot-http-server reflect-metadata pnpm add @dooboostore/simple-boot-http-server reflect-metadata npm init @dooboostore/simple-boot-http-server projectname

Features

Declarative Routing

Use decorators (@Router, @Route) to map URLs to classes and methods

데코레이터(@Router, @Route)를 사용하여 URL을 클래스와 메서드에 매핑

HTTP Method Mapping

Decorators for all standard HTTP methods (@GET, @POST, @PUT, @DELETE, etc.)

모든 표준 HTTP 메서드를 위한 데코레이터(@GET, @POST, @PUT, @DELETE 등)

Request/Response Handling

Unified RequestResponse object to manage HTTP transactions

HTTP 트랜잭션을 관리하는 통합된 RequestResponse 객체

Middleware/Filters

Intercept requests and responses for cross-cutting concerns

횡단 관심사를 위한 요청 및 응답 가로채기

Global Exception Handling

Centralized error handling using @Sim advice classes

@Sim advice 클래스를 사용한 중앙집중식 오류 처리

Session Management

Built-in session management with customizable storage providers

사용자 정의 저장소 제공자를 지원하는 내장 세션 관리

HTTPS Support

Configure the server to run over HTTPS and HTTP/2

HTTPS 및 HTTP/2를 통해 실행되도록 서버 구성

Endpoints

Define logic for different stages of request lifecycle

요청 생명주기의 다양한 단계에 대한 로직 정의

Quick Start

Basic API Setup

Create a simple REST API with HTTP method decorators:

HTTP 메서드 데코레이터를 사용하여 간단한 REST API를 생성하세요:

import { Sim, Router, Route, RouterModule } from '''@dooboostore/simple-boot'''; import { GET, POST, PUT, DELETE, Mimes, RequestResponse, ReqJsonBody } from '''@dooboostore/simple-boot-http-server'''; @Sim @Router({ path: '''/api/items''' }) export class ItemApi { @Route({ path: '''''' }) @GET({ res: { contentType: Mimes.ApplicationJson } }) getAllItems() { // Logic to get all items return [{ id: 1, name: '''Item 1''' }, { id: 2, name: '''Item 2''' }]; } @Route({ path: '''/{id}''' }) @GET({ res: { contentType: Mimes.ApplicationJson } }) getItemById(routerModule: RouterModule) { const itemId = routerModule.pathData?.id; // Logic to get item by ID return { id: itemId, name: `Item ${itemId}` }; } @Route({ path: '''''' }) @POST({ req: { contentType: [Mimes.ApplicationJson] }, res: { status: 201, contentType: Mimes.ApplicationJson } }) createItem(body: ReqJsonBody) { console.log('''Creating new item:''', body); // Logic to create a new item return { success: true, id: 3, ...body }; } @Route({ path: '''/{id}''' }) @PUT({ req: { contentType: [Mimes.ApplicationJson] } }) updateItem(routerModule: RouterModule, body: ReqJsonBody) { const itemId = routerModule.pathData?.id; console.log(`Updating item ${itemId}:`, body); // Logic to update an item return { success: true, id: itemId, ...body }; } @Route({ path: '''/{id}''' }) @DELETE({ res: { status: 204 } }) deleteItem(routerModule: RouterModule) { const itemId = routerModule.pathData?.id; console.log(`Deleting item ${itemId}`); // Logic to delete an item // No content returned for 204 } }

Server Setup

Bootstrap your HTTP server with the framework:

프레임워크로 HTTP 서버를 부트스트랩하세요:

import { SimpleApplication } from '''@dooboostore/simple-boot'''; import { SimpleBootHttpServer, HttpServerOption } from '''@dooboostore/simple-boot-http-server'''; // Create your main router @Sim @Router({ path: '''''', route: { '''/api''': ItemApi, // Add more routes here } }) class MainRouter {} // Configure and start the server const httpOption = new HttpServerOption({ port: 3000, host: '''localhost''' }); const app = new SimpleApplication({ rootRouter: MainRouter }); const server = new SimpleBootHttpServer(app, httpOption); server.run().then(() => { console.log('''Server is running on http://localhost:3000'''); });

Advanced Features

Filters & Middleware

Intercept requests and responses for cross-cutting concerns:

횡단 관심사를 위해 요청과 응답을 가로채세요:

import { Filter, RequestResponse, SimpleBootHttpServer } from '''@dooboostore/simple-boot-http-server'''; export class LoggingFilter implements Filter { async before(rr: RequestResponse, app: SimpleBootHttpServer): Promise { console.log(`[${new Date().toISOString()}] ${rr.reqMethod()} ${rr.reqUrl}`); return true; // Continue the chain } async after(rr: RequestResponse, app: SimpleBootHttpServer, sw: boolean): Promise { console.log(`Request finished with status ${rr.resStatusCode()}`); return true; } } // CORS Filter example export class CorsFilter implements Filter { async before(rr: RequestResponse, app: SimpleBootHttpServer): Promise { rr.resSetHeader('''Access-Control-Allow-Origin''', '''*'''); rr.resSetHeader('''Access-Control-Allow-Methods''', '''GET, POST, PUT, DELETE, OPTIONS'''); rr.resSetHeader('''Access-Control-Allow-Headers''', '''Content-Type, Authorization'''); if (rr.reqMethod() === '''OPTIONS''') { rr.resStatusCode(200).resEnd(); return false; // Stop the chain for preflight requests } return true; } } // Register filters in HttpServerOption const option = new HttpServerOption({ filters: [LoggingFilter, CorsFilter] });

Global Exception Handling

Handle errors globally with advice classes:

advice 클래스로 전역적으로 오류를 처리하세요:

import { Sim, ExceptionHandler, Inject, ExceptionHandlerSituationType } from '''@dooboostore/simple-boot'''; import { RequestResponse, NotFoundError, HttpError, InternalServerError, Mimes, HttpStatus } from '''@dooboostore/simple-boot-http-server'''; @Sim export class GlobalErrorHandler { @ExceptionHandler({ type: NotFoundError }) handleNotFound(rr: RequestResponse, e: NotFoundError) { // Critical: Check if response is already sent if (rr.resIsDone()) { console.error(`Not found error for URL: ${rr.reqUrl}, but response already sent.`, e); return; } rr.resStatusCode(404) .resSetHeader('''Content-Type''', Mimes.ApplicationJson); rr.resEnd(JSON.stringify({ error: '''Resource not found''' })); } @ExceptionHandler() async catch(@Inject({situationType: ExceptionHandlerSituationType.ERROR_OBJECT}) e: any, rr: RequestResponse) { console.error(`GlobalAdvice.catch ${rr.reqUrl}`, e); // Critical: Prevent ERR_HTTP_HEADERS_SENT error if (rr.resIsDone()) { return; } rr.resStatusCode(HttpStatus.InternalServerError); rr.resSetHeader('''Content-Type''', Mimes.ApplicationJson); let data = ''''''; if (e instanceof HttpError) { rr.resStatusCode(e.status); data = JSON.stringify(e); } else if (e instanceof Error) { const error = new InternalServerError(); error.data = { message: e.message }; data = JSON.stringify(error); } else { const error = new InternalServerError(); error.data = e; data = JSON.stringify(error); } await rr.resEnd(data); } } // Register in HttpServerOption const option = new HttpServerOption({ globalAdvice: GlobalErrorHandler, noSuchRouteEndPointMappingThrow: (rr) => new NotFoundError({ message: `No route for ${rr.reqUrl}` }), });

Session Management

Built-in session management with automatic cookie handling:

자동 쿠키 처리를 포함한 내장 세션 관리:

import { RequestResponse } from '''@dooboostore/simple-boot-http-server'''; @Sim @Router({ path: '''/api/session''' }) export class SessionApi { @Route({ path: '''/login''' }) @POST({ res: { contentType: Mimes.ApplicationJson } }) async login(rr: RequestResponse, body: ReqJsonBody) { const session = await rr.reqSession(); // Validate user credentials (simplified) if (body.username === '''admin''' && body.password === '''password''') { session.userId = '''user123'''; session.username = body.username; session.loginTime = new Date().toISOString(); return { success: true, message: '''Logged in successfully''' }; } else { rr.resStatusCode(401); return { success: false, message: '''Invalid credentials''' }; } } @Route({ path: '''/profile''' }) @GET({ res: { contentType: Mimes.ApplicationJson } }) async getProfile(rr: RequestResponse) { const session = await rr.reqSession(); if (!session.userId) { rr.resStatusCode(401); return { error: '''Not authenticated''' }; } let visitCount = session.visitCount || 0; visitCount++; session.visitCount = visitCount; return { userId: session.userId, username: session.username, loginTime: session.loginTime, visitCount: visitCount }; } @Route({ path: '''/logout''' }) @POST({ res: { contentType: Mimes.ApplicationJson } }) async logout(rr: RequestResponse) { const session = await rr.reqSession(); // Clear session data Object.keys(session).forEach(key => delete session[key]); return { success: true, message: '''Logged out successfully''' }; } }

Lifecycle Endpoints

Hook into different stages of the request lifecycle:

요청 생명주기의 다양한 단계에 연결하세요:

import { EndPoint, RequestResponse, SimpleBootHttpServer } from '''@dooboostore/simple-boot-http-server'''; @Sim class RequestLogger implements EndPoint { async endPoint(rr: RequestResponse, app: SimpleBootHttpServer) { console.log(`New request: ${rr.reqMethod()} ${rr.reqUrl} from ${rr.reqRemoteAddress}`); } } @Sim class ConnectionLogger implements EndPoint { async endPoint(rr: RequestResponse, app: SimpleBootHttpServer) { console.log(`Connection closed for ${rr.reqRemoteAddress}`); } } @Sim class ErrorLogger implements EndPoint { async endPoint(rr: RequestResponse, app: SimpleBootHttpServer) { console.log(`Error occurred for ${rr.reqUrl}`); } } // Register endpoints in HttpServerOption const option = new HttpServerOption({ requestEndPoints: [RequestLogger], // Run when request starts closeEndPoints: [ConnectionLogger], // Run when connection closes errorEndPoints: [ErrorLogger] // Run when error occurs });

@Router & @Route

Declarative routing system that maps URLs to classes and methods

URL을 클래스와 메서드에 매핑하는 선언적 라우팅 시스템

@GET, @POST, @PUT, @DELETE 및 기타 HTTP 메서드 데코레이터

RequestResponse

Unified object for handling HTTP requests and responses

HTTP 요청과 응답을 처리하는 통합 객체

Filters

Middleware system for intercepting requests and responses

요청과 응답을 가로채는 미들웨어 시스템

Global Exception Handling

Centralized error handling with @ExceptionHandler decorators
