Quick Start

File Operations

Handle file operations with the File class:

File 클래스로 파일 작업을 처리하세요:

import { FileUtils } from '@dooboostore/core-node/file/FileUtils'; import { ConvertUtils } from '@dooboostore/core-node/convert/ConvertUtils'; async function handleFileUpload(base64Data: string) { // Convert Base64 string to Buffer const imageBuffer = ConvertUtils.toBuffer(base64Data, { encoding: 'base64' }); // Write buffer to a temporary file const tempFile = await FileUtils.writeFile(imageBuffer, { originalName: 'my-image.png' }); console.log(`File created at: ${tempFile.path}`); console.log(`File size: ${tempFile.size} bytes`); // Move file to permanent location const permanentPath = `./uploads/${tempFile.fileName}`; await tempFile.move(permanentPath); console.log(`File moved to: ${tempFile.path}`); }

Buffer Conversion

Convert strings to buffers with various encoding options:

다양한 인코딩 옵션으로 문자열을 버퍼로 변환하세요:

import { ConvertUtils } from '@dooboostore/core-node/convert/ConvertUtils'; // Convert Base64 data URL to Buffer const base64Data = 'data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNkYAAAAAYAAjCB0C8AAAAASUVORK5CYII='; const buffer = ConvertUtils.toBuffer(base64Data, { encoding: 'base64' }); // Convert regular string to Buffer const textBuffer = ConvertUtils.toBuffer('Hello World'); // Convert Buffer back to string const text = ConvertUtils.toString(buffer, { encoding: 'utf-8' });

File System Utilities

Use comprehensive file system utilities:

포괄적인 파일 시스템 유틸리티를 사용하세요:

import { FileUtils } from '@dooboostore/core-node/file/FileUtils'; // Check if file exists const exists = FileUtils.existesSync('./myfile.txt', { existes: (path) => console.log(`File exists: ${path}`), noExistes: (path) => console.log(`File not found: ${path}`) }); // Create directory await FileUtils.mkdir('./uploads', { recursive: true }); // Read file const content = FileUtils.readSync('./myfile.txt'); // Write file FileUtils.write('Hello World', { path: './output.txt' }); // Append to file FileUtils.writeAppend('\nNew line', { path: './output.txt' }); // Delete file FileUtils.deleteFileSync('./temp.txt');

Advanced Features

File Class Methods

Use the File class for object-oriented file operations:

객체 지향 파일 작업을 위해 File 클래스를 사용하세요:

import { FileUtils } from '@dooboostore/core-node/file/FileUtils'; // Create a File instance const file = new FileUtils.File({ path: './documents/report.pdf', originalName: 'monthly-report.pdf' }); // Get file information console.log('File name:', file.fileName); console.log('Directory:', file.directory); console.log('Extension:', file.extension); console.log('Original name:', file.originalName); console.log('File size:', file.size); // Copy file await file.copy('./backup/report-copy.pdf'); // Rename file await file.rename('annual-report.pdf'); // Move file to different directory await file.move('./archive/reports/annual-report.pdf'); // Delete file await file.delete();

Path Utilities

Handle cross-platform path operations:

크로스 플랫폼 경로 작업을 처리하세요:

import { FileUtils } from '@dooboostore/core-node/file/FileUtils'; // Join path segments const fullPath = FileUtils.path(['documents', 'images', 'photo.jpg']); // Result: 'documents/images/photo.jpg' (Unix) or 'documents\\images\\photo.jpg' (Windows) // Array of path segments const pathSegments = ['home', 'user', 'projects', 'myapp']; const projectPath = FileUtils.path(pathSegments); // Single path string const singlePath = FileUtils.path('./src/components/Button.tsx');

Asynchronous Operations

Handle asynchronous file operations efficiently:

비동기 파일 작업을 효율적으로 처리하세요:

import { FileUtils } from '@dooboostore/core-node/file/FileUtils'; async function processFiles() { try { // Create directory asynchronously const dirPath = await FileUtils.mkdir('./temp/processing'); // Read file asynchronously const content = await FileUtils.read('./input.txt', { option: { encoding: 'utf-8' } }); // Process content and write to new file const processedContent = content.toString().toUpperCase(); const buffer = Buffer.from(processedContent); const outputFile = await FileUtils.writeFile(buffer, { path: './temp/processing/output.txt', originalName: 'processed-input.txt' }); console.log('Processing complete:', outputFile.path); } catch (error) { console.error('File processing error:', error); } }

API Reference

FileUtils Namespace

File Class

Object-oriented file wrapper with methods for file operations

파일 작업을 위한 메서드를 포함한 객체 지향 파일 래퍼

  • constructor(data: {path: string, originalName?: string})
  • copy(newPath: string): Promise<void>
  • move(newPath: string): Promise<void>
  • rename(newName: string): Promise<void>
  • delete(): Promise<void>
  • updateStats(): Promise<void>

Utility Functions

Static utility functions for common file operations

일반적인 파일 작업을 위한 정적 유틸리티 함수

  • writeFile(buffer: Buffer, config): Promise<File>
  • existesSync(path: PathParamType): boolean
  • mkdir(path: PathParamType): Promise<string>
  • readSync(path: PathParamType): Buffer
  • write(data: string | Buffer, config): string
  • deleteSync(path: PathParamType): void

ConvertUtils Namespace

String Conversion

Convert between strings and buffers with encoding support

인코딩 지원으로 문자열과 버퍼 간 변환

  • toString(data: string | Buffer, config?): string
  • toBuffer(data: string, config?): Buffer

Encoding Options

Support for various buffer encodings including Base64

Base64를 포함한 다양한 버퍼 인코딩 지원

  • encoding: BufferEncoding - utf-8, base64, hex, etc.
  • Automatic Base64 data URL prefix removal
