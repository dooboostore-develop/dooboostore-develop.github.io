Quick Start

Array Utilities

Powerful array manipulation functions:

강력한 배열 조작 함수들:

import { ArrayUtils } from '@dooboostore/core/array/ArrayUtils'; const myArray = [1, 2, 3, 4, 5]; // Shuffle the array const shuffled = ArrayUtils.toShuffle(myArray); console.log(shuffled); // Pick a random element const randomItem = ArrayUtils.pick(myArray); console.log(randomItem); // e.g., 3 // Pick 3 random unique elements const randomItems = ArrayUtils.pick(myArray, 3); console.log(randomItems); // e.g., [5, 1, 3] // Create 2D array const grid = ArrayUtils.create2DArray(3, 3, 0); console.log(grid); // [[0,0,0], [0,0,0], [0,0,0]] // Set operations const arr1 = [1, 2, 3, 4]; const arr2 = [3, 4, 5, 6]; const intersection = ArrayUtils.intersection(arr1, arr2); console.log(intersection); // [3, 4]

Date Utilities

Comprehensive date formatting and manipulation:

포괄적인 날짜 포맷팅과 조작:

import { DateUtils } from '@dooboostore/core/date/DateUtils'; const now = new Date(); // Format a date const formatted = DateUtils.format(now, 'yyyy-MM-dd HH:mm:ss'); console.log(formatted); // "2025-06-28 10:30:00" // Get relative time const fiveMinutesAgo = new Date(Date.now() - 5 * 60 * 1000); const relative = DateUtils.relativeTime(fiveMinutesAgo); console.log(relative); // "5 minutes ago" // Date calculations const tomorrow = DateUtils.addDays(now, 1); const nextWeek = DateUtils.addWeeks(now, 1); const nextMonth = DateUtils.addMonths(now, 1); // Compare dates const isAfter = DateUtils.isAfter(tomorrow, now); console.log(isAfter); // true

HTTP Client

Flexible HTTP client with JSON and XML support:

JSON과 XML을 지원하는 유연한 HTTP 클라이언트:

import { HttpJsonFetcher, HttpResponseError } from '@dooboostore/core/fetch/HttpJsonFetcher'; const jsonFetcher = new HttpJsonFetcher<any, any>(); async function fetchData() { try { // GET request for JSON data const user = await jsonFetcher.get({ target: `https://jsonplaceholder.typicode.com/users/1` }); console.log('Fetched User:', user); // POST request with JSON body const newProduct = await jsonFetcher.postJson({ target: `https://dummyjson.com/products/add`, config: { fetch: { body: { title: 'New Awesome Gadget', price: 199.99 }, }, }, }); console.log('Created Product:', newProduct); } catch (error: any) { if (error instanceof HttpResponseError) { console.error('HTTP Error:', error.message, error.response?.status); } else { console.error('Generic Error:', error); } } }

Observable & Operators

Reactive programming with RxJS-like operators:

RxJS 스타일 연산자를 사용한 반응형 프로그래밍:

import { Observable } from '@dooboostore/core/message/Observable'; import { Subject } from '@dooboostore/core/message/Subject'; import { map } from '@dooboostore/core/message/operators/map'; import { filter } from '@dooboostore/core/message/operators/filter'; import { debounceTime } from '@dooboostore/core/message/operators/debounceTime'; // Create a simple Observable const source$ = new Observable<number>(subscriber => { let count = 0; const intervalId = setInterval(() => { subscriber.next(count++); if (count > 5) subscriber.complete(); }, 100); return () => clearInterval(intervalId); }); // Transform and filter data source$.pipe( filter(num => num % 2 === 0), // Only even numbers map(num => `Number: ${num * 10}`) ).subscribe({ next: (data) => console.log(data), complete: () => console.log('Completed!') }); // Debounced search input const searchInput$ = new Subject<string>(); searchInput$.pipe( debounceTime(500) // Wait for 500ms of inactivity ).subscribe(searchText => { console.log('Searching for:', searchText); });

Advanced Features

CSS Parser

Parse and manipulate CSS programmatically:

CSS를 프로그래밍 방식으로 파싱하고 조작하세요:

import { CssParser, StyleRule, AtRule } from '@dooboostore/core/parser/css/CssParser'; const cssString = ` body { font-family: Arial, sans-serif; margin: 0; } @media (max-width: 600px) { .container { width: 100%; } } `; const parser = new CssParser(cssString); // Find and modify rules const bodyRule = parser.rules.find(rule => rule instanceof StyleRule && rule.selector === 'body' ) as StyleRule; if (bodyRule) { // Add new declaration bodyRule.content.push({ type: 'declaration', property: 'color', value: '#333' }); } // Add new rule parser.append('.footer { text-align: center; }'); console.log(parser.stringify());

2D/3D Geometry & Rendering

Comprehensive geometry classes with rendering capabilities:

렌더링 기능을 포함한 포괄적인 기하학 클래스:

import { Point2D, Vector, Rect, Polygon } from '@dooboostore/core/entity'; // 2D Points and Vectors const point1 = new Point2D(10, 20); const point2 = new Point2D(30, 40); const vector = new Vector(point1, point2); console.log(vector.magnitude()); // Distance between points console.log(vector.angle()); // Angle in radians // Rectangle operations const rect = new Rect(0, 0, 100, 50); const centerPoint = new Point2D(50, 25); console.log(rect.contains(centerPoint)); // true // Polygon operations const triangle = new Polygon([ new Point2D(0, 0), new Point2D(50, 0), new Point2D(25, 50) ]); const testPoint = new Point2D(25, 20); console.log(triangle.contains(testPoint)); // true // Collision detection const rect2 = new Rect(50, 25, 100, 50); console.log(rect.overlaps(rect2)); // true

Promise Utilities

Advanced Promise handling and state management:

고급 Promise 처리 및 상태 관리:

import { Promises } from '@dooboostore/core/promise/Promises'; // Sleep utility await Promises.sleep(1000); // Wait 1 second // Polling with loop const result = await Promises.loop( async () => { const response = await fetch('/api/status'); return response.json(); }, (data) => data.status === 'complete', // Continue until complete { interval: 1000, maxAttempts: 10 } ); // Transaction management const transaction = Promises.transaction(); try { const user = await transaction.add(() => createUser(userData)); const profile = await transaction.add(() => createProfile(user.id)); await transaction.commit(); } catch (error) { await transaction.rollback(); // Undo all operations } // Result wrapper for safe error handling const safeResult = await Promises.Result.wrap(riskyOperation()); if (safeResult.isSuccess()) { console.log('Success:', safeResult.data); } else { console.error('Error:', safeResult.error); }
