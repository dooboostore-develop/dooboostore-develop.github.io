core-web

Browser and web platform utilities for modern web applications

현대적인 웹 애플리케이션을 위한 브라우저와 웹 플랫폼 유틸리티

Installs the package and its peer dependency

패키지와 필수 의존성을 설치하세요:

npm install @dooboostore/core-web reflect-metadata pnpm add @dooboostore/core-web reflect-metadata

  • selectorElements(selector: string, element?: Element | Document): Element[] Select elements by selector 셀렉터로 요소 선택
  • selectorNodes(selector: string, element?: Element | Document): Node[] Select nodes by selector 셀렉터로 노드 선택

  • setAttribute(element: Element, attrs: string[]): void Set multiple attributes 여러 속성 설정
