simple-boot

Lightweight framework with DI and AOP support

DI, AOP를 지원하는 경량 프레임워크

Installation

Install the package and its peer dependency:

패키지와 필수 의존성을 설치하세요:

npm install @dooboostore/simple-boot reflect-metadata pnpm add @dooboostore/simple-boot reflect-metadata npm init @dooboostore/simple-boot projectname

Quick Start

Basic Usage

Here's a simple example to get you started:

시작하기 위한 간단한 예제입니다:

import 'reflect-metadata'; import { SimpleApplication, Sim, Router } from '@dooboostore/simple-boot'; @Sim // Decorator can be used without parentheses class Greeter { say() { console.log('Hello, Simple Boot!'); } } @Sim @Router({ path: '', route: { '/user': Greeter } }) class AppRouter {} const app = new SimpleApplication({ rootRouter: AppRouter }); app.run(); // Get instance and call method const greeter = app.sim(Greeter); greeter?.say(); // Or, route to the module and execute app.routing('/user').then(routerModule => { routerModule.getModuleInstance<Greeter>()?.say(); });

Dependency Injection

Simple Boot automatically handles dependency injection based on constructor parameter types:

Simple Boot은 생성자 매개변수 타입을 기반으로 의존성 주입을 자동으로 처리합니다:

@Sim // Using without parentheses for default settings class ProjectService { sum(x: number, y: number): number { return x + y; } } @Sim() // Using with parentheses is also valid class User { constructor(private projectService: ProjectService) {} calculate() { const result = this.projectService.sum(5, 25); console.log(`Calculation result is: ${result}`); } } // somewhere in your app const user = app.sim(User); user?.calculate(); //-> Calculation result is: 30

Lifecycle Management

Control object lifecycle with Singleton and Transient scopes:

Singleton과 Transient 스코프로 객체 생명주기를 제어하세요:

import { Sim, Lifecycle } from '@dooboostore/simple-boot'; // Singleton (Default) - Same instance every time @Sim({ scope: Lifecycle.Singleton }) class DatabaseService { private connection = 'db-connection'; getConnection() { return this.connection; } } // Transient - A new instance is created on each request @Sim({ scope: Lifecycle.Transient }) class RequestHandler { private requestId = Math.random(); getId() { return this.requestId; } }

AOP Example

Use AOP decorators to intercept method calls:

AOP 데코레이터를 사용하여 메서드 호출을 가로채세요:

import { Sim, Before, After, Around, AroundForceReturn } from '@dooboostore/simple-boot'; const LoggingAround = () => Around({ before: (obj, propertyKey, args) => { console.log(`[${propertyKey}] Before with args:`, args); // You can modify arguments here return args.map(arg => typeof arg === 'number' ? arg * 2 : arg); }, after: (obj, propertyKey, args, beforeReturn) => { console.log(`[${propertyKey}] After with result:`, beforeReturn); // You can force a different return value if (beforeReturn > 10) { throw new AroundForceReturn("Value is too high!"); } return beforeReturn; } }); @Sim class Calculator { @Before({ property: 'add' }) logBefore() { console.log('Preparing to add...'); } @After({ property: 'add' }) logAfter() { console.log('Addition complete.'); } @LoggingAround() add(a: number, b: number): number { console.log('Executing add method'); return a + b; } }

Advanced Features

Exception Handling

Handle exceptions locally or globally with @ExceptionHandler:

@ExceptionHandler로 예외를 지역적 또는 전역적으로 처리하세요:

class MyError extends Error {} @Sim class UserService { @ExceptionHandler({ type: MyError }) handleMyError(e: MyError) { console.error(`Caught a specific MyError: ${e.message}`); } @ExceptionHandler() handleAllOtherErrors(e: any) { console.error(`Caught a generic error:`, e); } doSomething() { throw new MyError('Something went wrong!'); } }

Caching

Cache method results with @Cache decorator:

@Cache 데코레이터로 메서드 결과를 캐시하세요:

@Sim class UserService { @Cache({ key: (id: string) => `user:${id}`, ms: 5000 }) // Dynamic key, 5-second TTL getUserById(id: string) { console.log(`Fetching user ${id} from DB...`); return { id, name: 'User ' + id }; } @Cache({ deleteKey: (id: string) => `user:${id}` }) updateUser(id: string, data: any) { console.log(`Updating user ${id}, cache will be cleared.`); // ... update logic } }

Validation

Validate class properties with decorators:

데코레이터로 클래스 속성을 검증하세요:

import { Sim, Validation, NotEmpty, NotNull, Regexp, execValidationInValid, ValidException } from '@dooboostore/simple-boot'; @Sim class UserProfile { @Validation(NotEmpty) @Validation(Regexp(/^[a-zA-Z]+$/)) name: string = ''; @Validation(NotNull) age: number | null = null; } const profile = new UserProfile(); profile.name = '123'; // Invalid name profile.age = null; // Invalid age const validationErrors = execValidationInValid(profile); if (validationErrors.length > 0) { throw new ValidException(validationErrors); }

Core Concepts

@Sim

Core decorator that marks a class for management by the Simple Boot container

Simple Boot 컨테이너에서 관리할 클래스를 표시하는 핵심 데코레이터

SimpleApplication

Application entry point that initializes and manages the DI container

DI 컨테이너를 초기화하고 관리하는 애플리케이션 진입점

@Router & @Route

Routing system that maps URL paths to classes or methods

URL 경로를 클래스나 메서드에 매핑하는 라우팅 시스템

@Before/@After/@Around

AOP decorators for intercepting method calls

메서드 호출을 가로채는 AOP 데코레이터

Intent System

Event system for decoupled communication between components

컴포넌트 간 분리된 통신을 위한 이벤트 시스템

@Cache

Method-level caching with TTL and dynamic key support

TTL과 동적 키를 지원하는 메서드 레벨 캐싱
