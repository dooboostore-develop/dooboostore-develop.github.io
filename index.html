What's in dooboostore?

Everything you need to build great applications.

dooboostore에는 무엇이 있나요?

훌륭한 애플리케이션을 만들기 위해 필요한 모든 것.

🚀

Dependency Injection

Powerful DI container with AOP support for clean, maintainable code architecture.

의존성 주입

깔끔하고 유지보수 가능한 코드 아키텍처를 위한 AOP 지원 강력한 DI 컨테이너.

⚡

Rich Utilities

Comprehensive utility library with array, object, string, date manipulation and more.

풍부한 유틸리티

배열, 객체, 문자열, 날짜 조작 등을 포함한 포괄적인 유틸리티 라이브러리.

🔧

DOM Rendering, Parsing

Lightweight DOM parsing, rendering engine with reactive data binding and component lifecycle.

DOM 렌더링

반응형 데이터 바인딩과 컴포넌트 생명주기를 가진 경량 DOM 렌더링 엔진.

📦

HTTP Server

Build robust HTTP servers with decorative routing, middleware, and session management.

HTTP 서버

데코레이터 라우팅, 미들웨어, 세션 관리로 견고한 HTTP 서버를 구축하세요.

🎨

Observable System

RxJS-like reactive programming with operators for handling asynchronous data streams.

Observable 시스템

비동기 데이터 스트림 처리를 위한 연산자가 있는 RxJS와 같은 반응형 프로그래밍.

🔄

Frontend Framework

Single Page Application framework with component-based architecture and routing.

프론트엔드 프레임워크

컴포넌트 기반 아키텍처와 라우팅을 가진 단일 페이지 애플리케이션 프레임워크.

🛠️

Type Safety

Built with TypeScript-first approach providing comprehensive type definitions and safety.

타입 안전성

포괄적인 타입 정의와 안전성을 제공하는 TypeScript 우선 접근 방식으로 구축.

⚙️

Modular Architecture

Pick and choose only the packages you need with a fully modular ecosystem design.

모듈형 아키텍처

완전히 모듈화된 생태계 설계로 필요한 패키지만 선택해서 사용하세요.

User Tracking (SWT)

Lightweight user tracking withSimpleWebTracker (SWT). Analyze user behavior with session replays, heatmaps, and a built-in dashboard.

사용자 추적 (SWT)

경량 사용자 추적 라이브러리SimpleWebTracker (SWT). 세션 리플레이, 히트맵, 내장 대시보드로 사용자 행동을 분석하세요.
